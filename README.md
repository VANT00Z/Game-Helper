# Game Helper

Ваш помощник в играх!

# Описание

Мини википедия, браузер и калькулятор в одном приложении!
Множество тонкостей игр, автономный калькулятор с различными раширениями, удобный браузер, чтобы вы не запутались в массивах данных.

# Технологии

- Python 3.8+
- HTML 5
- CSS 3

# Установка и запуск
## Требования

- Windows 7+
- 2 ГБ ОЗУ+
- 1 ГГц CPU+
- 50 МБ места на диске

## Запуск

Для запуска приложения запустите [Game Helper.exe](Game-Helper.exe)
